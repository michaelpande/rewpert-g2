What is this?

This plugin was made by Michael Pande, Petter L. Olsen and Diego Alonso Pasten Bahamondes. It is a restful plugin that will extract content from NewsML-G2 documents and then publish it directly to WordPress

Getting started

Requirements

WordPress

PHP 5.3

Set up on WordPress

Click on the 'Install on WordPress' button on the top of this site. This will let install the plugin without hassle if you're logged in.

How to find your API-key

Our magnificent coders have made it super easy for you. Just go to the Admin panel in WordPress. Choose Posts -> RESTful NewsML-G2 API plugin. There will be a long link there. Copy that. This is your key
